# 📊 Sales Data Analysis & Excel Dashboard Project

## 🚀 Project Overview

This project focuses on analyzing a real-world sales dataset using **Microsoft Excel**.
It includes data cleaning, advanced Excel functions, statistical analysis, and an interactive dashboard to generate meaningful business insights.

---

## 📁 Dataset Details

The dataset contains the following fields:

* Customer Name
* Region (West, East, North, South, Central)
* Product Category (Books, Clothing, Electronics, Furniture, Office Supplies)
* Sales
* Quantity
* Discount
* Order Date
* Profit

---

## 🎯 Objectives

* Analyze sales performance across regions and categories
* Identify high-value customers
* Understand the impact of discounts on profit
* Generate insights using statistical tools
* Build an interactive Excel dashboard

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Data Analysis Toolpak
* Excel Formulas (INDEX, MATCH, IF, etc.)
* Charts & Data Visualization

---

## 📌 Key Tasks Performed

### 1️⃣ Conditional Formatting

* Highlighted **Top 10 customers** based on total sales
* Used Excel's built-in **Top/Bottom Rules**

---

### 2️⃣ WHAT-IF Analysis

* Used **Data Tables / Scenario Manager**
* Analyzed how changing **discount (%) impacts total profit**

---

### 3️⃣ Linear Regression (Profit vs Sales)

* Used **Data Analysis Toolpak → Regression**
* Identified relationship between **Sales and Profit**

---

### 4️⃣ Descriptive Statistics

* Generated statistical summary:

  * Mean
  * Median
  * Standard Deviation
  * Min / Max

---

### 5️⃣ Monthly Sales Growth

* Calculated month-over-month growth:

```excel
=(Current Month - Previous Month)/Previous Month
```

* Added **▲ / ▼ arrows** using conditional formatting

---

### 6️⃣ Timestamp Column

* Created dynamic timestamp using:

```excel
=NOW()
```

---

### 7️⃣ High-Value Customers Identification

Used:

```excel
=INDEX()
=MATCH()
=FILTER()
```

* Extracted customers with highest purchase value

---

### 8️⃣ Pivot Tables

Created Pivot Tables for:

* Sales by Region
* Sales by Product Category
* Profit Analysis

---

### 9️⃣ Data Visualization

Built multiple charts:

* 📈 Line Chart → Sales Trend
* 📊 Bar Chart → Region-wise Sales
* 🥧 Pie Chart → Category Contribution

---

### 🔟 Dashboard Creation

Developed an interactive dashboard including:

* KPI Cards (Sales, Profit, Orders, Discount)
* Sales Trend Analysis
* Region & Category Insights
* Top Customers
* Slicers for filtering data

---

## 📊 Dashboard Features

* Fully interactive with slicers
* Clean and professional layout
* Dynamic charts updating with filters
* Business insights at a glance

---

## 💡 Key Insights

* Certain regions contribute significantly more to total sales
* High discounts can reduce profitability
* A small group of customers generate a large portion of revenue
* Seasonal trends impact sales performance
